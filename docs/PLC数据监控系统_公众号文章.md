# 告别"盲操作"：一款开源免费的 PLC 数据监控工具，让产线状态一目了然

> 适配西门子全系列 S7 PLC，支持 Windows / Linux 跨平台运行，本地 SQLite + 远程 MySQL 双保险数据存储

---

## 引言

在工业自动化领域，PLC（可编程逻辑控制器）是产线设备的"大脑"。然而，很多中小工厂的工程师和操作人员面临一个共同痛点：**PLC 里的数据看不见、摸不着**——要么只能通过 PLC 厂家昂贵的上位机软件查看，要么只能在设备面板上盯几个数字。

今天介绍的这款 **PLC 数据监控系统（PLCMonitor）**，就是为解决这个问题而生。它是一款基于 Qt 6 开发的跨平台桌面应用，通过 **Snap7 开源协议栈**直接与西门子 S7 系列 PLC 通信，实现**实时数据采集、可视化展示、历史存储与趋势分析**，帮助工程师高效监控生产现场。

---

## 核心功能一览

### 实时数据采集 — 零延迟感知产线状态

PLCMonitor 通过 Snap7 协议直接读取 PLC 的 DB 块数据，支持 BOOL、INT、REAL 等全类型标签。可根据需要配置轮询间隔（100ms ~ 10s），在实时性和系统负载间灵活平衡。

![功能示意图]

**支持数据类型**：BOOL / BYTE / WORD / DWORD / INT / DINT / REAL / STRING

---

### 四大视图模式 — 想怎么看，就怎么看

| 视图 | 适用场景 |
|------|----------|
| **工艺总貌** | 图形化展示生产流程，设备状态一目了然 |
| **实时数据** | 表格形式展示所有监控标签的当前值 |
| **趋势图表** | 折线图实时展示数据变化趋势（8条曲线同屏） |
| **历史数据** | 查询任意时间段的历史趋势，支持数据导出 |

**一个细节**：标签管理器会跟随主视图自动切换，工艺视图、实时表格、趋势图表的标签选择各自独立保存，切换无需重复勾选。

---

### 数据持久化 — 双保险存储策略

- **本地 SQLite**：所有数据实时写入本地数据库，不需网络即可独立运行
- **远程 MySQL**（可选）：配置后自动同步至远程数据库，支持多客户端数据共享
- **多粒度聚合**：自动生成分钟/小时/天级别的聚合数据，历史查询秒级响应

## 快速上手

### 1. 配置 PLC 连接

编辑 `configs/plc_config.json`：

```json
{
  "plc": {
    "ip": "192.168.0.11",
    "rack": 0,
    "slot": 1
  }
}
```

### 2. 配置监控标签

```json
{
  "tags": [
    {
      "name": "温度传感器1",
      "dbNumber": 1,
      "byteOffset": 2,
      "bitOffset": -1,
      "dataType": "REAL",
      "writable": false
    }
  ]
}
```

### 3. 启动程序，自动连接 PLC

启动后 1 秒自动连接（可关闭），数据即刻呈现在界面上。
